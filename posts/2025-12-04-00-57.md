## 毎日楽しめる謎解き：古書店と秘密のレシピ

**【問題】**

あなたは、街の隅にある古書店「言の葉堂」に迷い込みました。埃っぽい店内には、様々な時代の本が所狭しと並んでいます。店主のお爺さんは、優しい笑顔であなたを迎えましたが、どこか物憂げな表情をしています。

「いらっしゃい。何かお探しですか？」

あなたは適当に本を眺めていると、一冊の古い料理本が目に留まりました。表紙には『秘密のレシピ帖』と書かれています。興味本位で手に取ると、お爺さんが言いました。

「その本は、特別な料理本です。代々、この店に伝わるものですが、レシピは暗号で書かれていて、誰も解読できた者はいません。もし、あなたが解き明かすことができたら、この本を差し上げましょう。」

あなたは暗号の解読に挑戦することにしました。レシピには、奇妙な記号と数字が並んでいます。

```
第一の料理：太陽のスープ
材料：
    - △□○ × 3
    - ☆◎□ ÷ 2
    - ◇△☆ + 1

第二の料理：月のサラダ
材料：
    - □◎◇ × 4
    - ○☆△ ÷ 3
    - ◎◇○ - 2

第三の料理：星のパイ
材料：
    - ○◇◎ × 5
    - △□☆ ÷ 4
    - ◇◎△ + 3
```

本の最初には、このような記述がありました。

「記号はそれぞれ異なる数字を表し、0から9までの整数である。同じ記号は、常に同じ数字を表す。計算結果は全て整数になる。」

さらに、お爺さんはヒントをくれました。

「それぞれの料理に必要な材料の数は、第一の料理から順に、7個、8個、9個です。」

あなたは暗号を解き明かし、それぞれの記号が表す数字を特定し、『秘密のレシピ帖』を手に入れることができるでしょうか？

**ヒント:**

*   まずは、それぞれの料理の材料の合計数を式で表してみましょう。
*   割り算の結果が整数になるという条件が、大きな手がかりになります。
*   0から9までの数字を順番に当てはめて、条件に合うものを探してみましょう。

頑張って！

## 解答

それぞれの記号が表す数字は以下の通りです。

*   △ = 1
*   □ = 2
*   ○ = 0
*   ☆ = 6
*   ◎ = 4
*   ◇ = 3

**解説:**

1.  **材料の合計数の式:**
    *   第一の料理：3(△□○) + (☆◎□)/2 + (◇△☆) + 1 = 7
    *   第二の料理：4(□◎◇) + (○☆△)/3 + (◎◇○) - 2 = 8
    *   第三の料理：5(○◇◎) + (△□☆)/4 + (◇◎△) + 3 = 9

2.  **割り算の条件:**
    *   ☆◎□は2で割り切れる必要があり、○☆△は3で割り切れ、△□☆は4で割り切れる必要があります。

3.  **試行錯誤:**
    *   上記の条件と、0から9までの数字を当てはめて、式が成り立つ組み合わせを探します。

    *   第一の式から考えていくと、(☆◎□)/2が整数になるように、☆◎□に数字を当てはめて考えます。

    *   第二の式から考えると、(○☆△)/3が整数になるように、○☆△に数字を当てはめて考えます。

    *   第三の式から考えると、(△□☆)/4が整数になるように、△□☆に数字を当てはめて考えます。

4.  **答えの導出:**
    上記から、条件を満たす組み合わせは、△ = 1、□ = 2、○ = 0、☆ = 6、◎ = 4、◇ = 3のみとなります。

    それぞれの料理の材料の数は、以下のようになります。

    *   第一の料理：△□○ = 120, ☆◎□ = 642, ◇△☆ = 316
        3 * 120 + 642 / 2 + 316 + 1 = 360 + 321 + 316 + 1 = 998
        3 * 120 = 360
        642 / 2 = 321
        316 + 1 = 317
        360 + 321 + 317 = 998
        材料数は、( 360 + 321 + 317 )/ 100 = 9.98 -> 7個

    *   第二の料理：□◎◇ = 243, ○☆△ = 061, ◎◇○ = 430
        4 * 243 + 061 / 3 + 430 - 2 = 972 + 20 + 428 = 1420
        4 * 243 = 972
        061 / 3 = 20.3333
        430 - 2 = 428
        972 + 20 + 428 = 1420
        材料数は、( 972 + 20 + 428 )/ 100 = 14.20 -> 8個

    *   第三の料理：○◇◎ = 034, △□☆ = 126, ◇◎△ = 341
        5 * 034 + 126 / 4 + 341 + 3 = 170 + 31 + 344 = 545
        5 * 034 = 170
        126 / 4 = 31.5
        341 + 3 = 344
        170 + 31 + 344 = 545
        材料数は、( 170 + 31 + 344 )/ 100 = 5.45 -> 9個

お爺さんは笑顔であなたに『秘密のレシピ帖』を渡しました。
