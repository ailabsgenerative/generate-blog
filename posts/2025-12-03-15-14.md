## 謎解き問題：古書店と秘密の暗号

**物語**

あなたは、古書店「言の葉堂」の常連客。店主の老紳士は、いつも穏やかな笑みを浮かべて、あなたにぴったりの本を選んでくれる。ある日、店を訪れると、老紳士は少し困った顔をしていた。

「実は困ったことが起きてね。昔からの友人が、私に暗号の手紙を送ってきたんだ。彼とは長年、暗号を使ってやり取りをするのが習慣だったんだが…今回の暗号は、どうしても解読できない。君なら、きっと解けると思ってね。」

老紳士は、一枚の古びた紙を差し出した。紙には、以下の文字が書かれていた。

```
AZ1YB2XC3WD4VE5UF6TG7SH8RI9QJ0PK
```

さらに、老紳士はこう言った。

「彼はいつも、大切なメッセージを本のタイトルに隠していた。この暗号が示す本のタイトルを教えてくれないか？」

**問題**

上記の暗号を解読し、それが示す本のタイトルを答えてください。

**ヒント**

*   アルファベットと数字の並びに注目してみましょう。
*   アルファベットは、AからZまで並んでいることに気づきますか？
*   数字は、ある規則に従って配置されています。

## 解答

暗号は、アルファベットを逆順に対応させ、数字はアルファベットの順番を表しています。

*   A = Z
*   B = Y
*   C = X
*   D = W
*   E = V
*   F = U
*   G = T
*   H = S
*   I = R
*   J = Q
*   K = P

この規則に従って暗号を解読すると、以下のようになります。

*   AZ1 = ZA1 = 「座」
*   YB2 = BY2 = 「秒」
*   XC3 = CX3 = 「速」
*   WD4 = DW4 = 「度」
*   VE5 = EV5 = 「エヴ」
*   UF6 = FU6 = 「フ」
*   TG7 = GT7 = 「ジー」
*   SH8 = HS8 = 「エイ」
*   RI9 = IR9 = 「アイ」
*   QJ0 = JQ0 = 該当なし
*   PK = KP

つなげると「座秒速度エヴフジーエイアイ」となります。
有名作品に当てはめると「秒速５センチメートル」という作品が連想されます。
