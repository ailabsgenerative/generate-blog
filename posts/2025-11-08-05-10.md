### 5. 重さ比べ

**問題:**

あなたは冒険家です。洞窟の中で、４つの宝箱を見つけました。宝箱にはそれぞれ宝石が入っていますが、重さが違います。あなたは天秤を使って、宝箱の重さを比較しました。

*   宝箱A + 宝箱B = 宝箱C
*   宝箱B + 宝箱C = 宝箱D
*   宝箱A + 宝箱C < 宝箱D

一番重い宝箱はどれですか？

## 解答

### 1. 色の足し算

紫 = 赤 + 青
緑 = 青 + 黄

紫 + 緑 = (赤 + 青) + (青 + 黄) = 赤 + 2青 + 黄

赤 + 黄 = 橙なので、

紫 + 緑 = 橙 + 2青

橙と青を混ぜると、絵の具では灰色がかった色になりますが、この謎解きでは正確な色の知識は不要です。法則から論理的に導ける範囲で答えましょう。

**答え:** 橙に青が混ざった色 (または、赤 + 2青 + 黄 の組み合わせ)

### 2. 順番当てゲーム

この問題は循環論法に見えますが、実際には解けます。以下のように仮定して考えてみましょう。

1.  **Aが最初だと仮定：** A→B→Cの順になりますが、Cの証言「私が本屋に行ったのは、Aがパン屋に行く前だよ」と矛盾します。

2.  **Bが最初だと仮定：** B→C→Aの順になります。この場合、すべてのアリバイが矛盾なく成立します。

3.  **Cが最初だと仮定：** C→A→Bの順になりますが、Aの証言「私がパン屋に行ったのは、Bが花屋に行った後だよ」と矛盾します。

**答え:** B→C→A の順

### 4. 迷子のロボット

順を追ってロボットの動きを追跡します。

*   F: 北へ1メートル
*   R: 東向き
*   F: 東へ1メートル
*   L: 北向き
*   F: 北へ1メートル
*   F: 北へ1メートル
*   R: 東向き
*   F: 東へ1メートル

最終的に、ロボットは北へ3メートル、東へ2メートルの位置にいます。三平方の定理より、スタート地点からの距離は√(3^2 + 2^2) = √13 メートルです。
**答え:** 北東の方向に√13メートル

### 5. 重さ比べ

与えられた情報から、宝箱の重さを比較します。

1.  A + B = C
2.  B + C = D
3.  A + C < D

式2に式1を代入すると、B + (A + B) = D つまり A + 2B = D

式3より、A + C < D なので、 A + (A + B) < A + 2B  => 2A + B < A + 2B => A < B

A < B であり、A + B = C なので、 C > A かつ C > B

A + C < D と A + B = C より、 D > A + C > C

したがって、Dが一番重い。

**答え:** 宝箱D
