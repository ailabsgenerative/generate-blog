## 毎日楽しめる謎解き：秘密のレシピを解読せよ！

**【問題】**

あなたは有名な料理研究家、味沢（あじさわ）先生の助手です。先生は世界中の珍しい食材を使った、秘密のレシピノートを持っています。しかし、先生は高齢のため、最近少しボケてきてしまい、レシピの重要な部分を暗号にしてしまいました。

今日、あなたは先生から新しいレシピの解読を頼まれました。レシピには、ある特別なスープの材料が書かれているのですが、肝心の材料名が以下のように暗号化されています。

```
材料：
・A → トマト
・B → タマネギ
・C → ニンジン
・D → ジャガイモ
・E → キャベツ
・F → 鶏肉
・G → 牛肉
・H → 豚肉
・I → 魚介
・J → キノコ
・K → 海藻

スープの材料：
1.  B + A + C
2.  D + D + E
3.  G + J
4.  I + K
5.  A + H + A

```

それぞれの行は、スープに入れる材料の組み合わせを表しています。
例えば、"B + A + C" は「タマネギとトマトとニンジン」という意味になります。

さて、この暗号を解読して、スープに使われている材料をすべて答えてください。

**【ヒント】**

特にありません。暗号表をよく見て、落ち着いて解読しましょう。

## 解答

スープの材料は以下の通りです。

1.  タマネギ、トマト、ニンジン
2.  ジャガイモ、ジャガイモ、キャベツ（ジャガイモ2つとキャベツ）
3.  牛肉、キノコ
4.  魚介、海藻
5.  トマト、豚肉、トマト（トマト2つと豚肉）
