## 毎日謎解き：秘密の宝箱

あなたは有名な冒険家、インディー・ジョーンズの孫として、祖父の残した手紙と古い宝箱を見つけました。手紙にはこう書かれています。

「我が孫よ、この宝箱には秘密の宝が隠されている。しかし、開けるためには3つの試練を乗り越えねばならない。知恵と勇気、そして何よりも論理的思考力を持って挑むが良い。」

宝箱には３つの鍵穴があり、それぞれに奇妙な装置が取り付けられています。

**試練１：色の迷路**

最初の鍵穴には、４色のボタン（赤、青、黄、緑）がランダムに配置された迷路のようなパネルが取り付けられています。パネルの上には以下の指示が書かれています。

1.  最初に好きな色のボタンを１つ押してください。
2.  次に、最初に押したボタンの色と隣り合う色のボタンを押してください。（隣り合うとは、上下左右に隣接しているボタンのことです。斜めは含みません。）
3.  同じ色のボタンを連続して押すことはできません。
4.  全てのボタンを一度ずつ押すと、鍵が開きます。

パネルのボタン配置は以下の通りです。

```
赤　青　黄
緑　赤　青
黄　緑　赤
```

**試練２：数字の階段**

２番目の鍵穴には、数字が書かれた階段状のブロックが並んでいます。階段は５段あり、各段には数字が書かれています。

```
　　　　　　５
　　　　　　３　８
　　　　　　１　６　９
　　　　　　２　４　７　５
　　　　　　？　？　？　？　？
```

最下段の数字は隠されています。この階段には法則があり、各段の数字は、その上の段の数字を使って計算されています。法則を見破り、最下段の数字を左から順に答えてください。

**試練３：論理の宝石**

３番目の鍵穴には、３つの宝石（ルビー、サファイア、エメラルド）が埋め込まれた台座があります。台座には、以下の３つのヒントが刻まれています。

1.  ルビーは真実を語る。
2.  サファイアは嘘をつく。
3.  エメラルドは気まぐれで、真実を言うこともあれば嘘を言うこともある。

あなたは各宝石に１つずつ質問をして、正しい宝石の配置を見つけなければなりません。しかし、質問は宝箱を開けるための正しい順番についてのみです。

あなたは３つの宝石にそれぞれ以下の質問をしました。

*   ルビー：「一番最初に押す宝石はどれですか？」
*   サファイア：「二番目に押す宝石はどれですか？」
*   エメラルド：「三番目に押す宝石はどれですか？」

それぞれの答えは以下の通りでした。

*   ルビー：「サファイア」
*   サファイア：「ルビー」
*   エメラルド：「ルビー」

これらの情報から、正しい宝石を押す順番を答えてください。

## 解答

**試練１：色の迷路**

この問題は、いくつかの解き方があります。

*   **例1:** 緑→赤→青→黄→赤→緑→黄
*   **例2:** 黄→赤→青→緑→黄→赤→緑

**試練２：数字の階段**

法則は、上の段の隣り合う数字を足し合わせるというものです。

*   ３＋８＝１１
*   １＋６＝７、６＋９＝１５
*   ２＋４＝６、４＋７＝１１、７＋５＝１２

したがって、最下段の数字は左から順に、

*   ７＋１５＝**２２**
*   １＋６＝**８**、６＋９＝**２２**
*   ２＋４＝**６**、４＋７＝**１１**、７＋５＝**１２**
*   ６＋１１＝**１７**、１１＋１２＝**２３**
*   １２＋２３＝**３５**

答え：**８、７、１５、１１、１２**

**試練３：論理の宝石**

この問題は、それぞれの宝石の発言が真実か嘘かを仮定して考えることで解けます。

*   **ルビーが真実を言っている場合:** ルビーの答えが「サファイア」なので、最初に押す宝石はサファイアです。サファイアは嘘を言っているので、「ルビー」という答えは嘘です。エメラルドは気まぐれなので、「ルビー」という答えは真実の場合も嘘の場合もあります。
*   **サファイアが嘘を言っている場合:** サファイアの答えが「ルビー」なので、二番目に押す宝石はルビーではありません。
*   **エメラルドが真実を言っている場合:** エメラルドの答えが「ルビー」なので、最後に押す宝石はルビーです。

これらの情報を整理すると、以下のような表を作成できます。

| 宝石      | 真実/嘘  | 発言         | 宝石の順番 |
| --------- | -------- | ------------ | -------- |
| ルビー    | 真実     | サファイア     | 1        |
| サファイア | 嘘       | ルビー       | 2        |
| エメラルド | 気まぐれ | ルビー       | 3        |

結論として、宝石を押す順番は「サファイア、エメラルド、ルビー」となります。
