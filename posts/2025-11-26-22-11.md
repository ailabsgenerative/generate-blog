## 毎日楽しめる謎解き：消えた王冠と秘密のレシピ

**物語**

ここは、美味しいお菓子と美しい景色で有名な小さな国、甘味（かんみ）王国。ところが最近、国中がなんだか落ち着かない様子。なぜなら、王室に代々伝わる大切な王冠が盗まれてしまったから！

王様は名探偵であるあなたに、王冠の捜索を依頼しました。手がかりは、王冠が盗まれた日の晩餐会で王様が飲もうとしていた、特別なハーブティーのレシピだけ。レシピには、材料のハーブの名前と、その分量が書かれています。

しかし、よく見るとレシピには奇妙な暗号が隠されているようです。どうやら、犯人は王冠を隠した場所を、このレシピに隠したようなのです。

**問題**

以下のハーブティーのレシピに隠された暗号を解き、王冠が隠された場所を特定してください。

**王室秘伝 ハーブティー レシピ**

*   カモミール：12g
*   ラベンダー：18g
*   ローズマリー：21g
*   ミント：13g
*   レモングラス：9g
*   バジル：2g

**ヒント**

1.  それぞれのハーブ名の文字数を数えてみましょう。
2.  レシピに書かれたハーブの分量と、ハーブ名の文字数には何か関係があるでしょうか？
3.  計算結果をアルファベットに置き換えてみましょう。アルファベットは、A=1, B=2, C=3…とします。
4.  導き出されたアルファベットを並び替えてみましょう。

## 解答

1.  ハーブ名の文字数を数えます。
    *   カモミール：6文字
    *   ラベンダー：5文字
    *   ローズマリー：6文字
    *   ミント：3文字
    *   レモングラス：5文字
    *   バジル：3文字
2.  ハーブの分量を文字数で割ります。
    *   カモミール：12g / 6文字 = 2
    *   ラベンダー：18g / 5文字 = 3あまり3（切り捨てて3）
    *   ローズマリー：21g / 6文字 = 3あまり3（切り捨てて3）
    *   ミント：13g / 3文字 = 4あまり1（切り捨てて4）
    *   レモングラス：9g / 5文字 = 1あまり4（切り捨てて1）
    *   バジル：2g / 3文字 = 0あまり2（切り捨てて0）
3.  計算結果をアルファベットに置き換えます。
    *   2 → B
    *   3 → C
    *   3 → C
    *   4 → D
    *   1 → A
    *   0 → A
4.  アルファベットを並び替えます。

導き出されたアルファベットは「BCCDAA」。これを並び替えると「BACKYARD」となり、王冠は裏庭（バックヤード）に隠されていることがわかります。
