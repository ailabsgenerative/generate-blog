## 毎日楽しめる謎解き問題集

**第1問：魔法の鏡**

あなたは魔法使いの弟子。師匠から「真実しか映さない鏡」を渡されました。しかし、その鏡は普段は曇っていて何も映りません。以下の３つの言葉を鏡に向かって言うと、どれか一つだけが鏡をクリアにし、真実を映し出します。

1.  「私は嘘つきだ」
2.  「今日は晴れている」
3.  「この謎は簡単だ」

どの言葉を言えば鏡はクリアになるでしょうか？

**第3問：盗まれた宝石**

あなたは名探偵。ある富豪の家で宝石が盗まれました。容疑者は３人。それぞれアリバイを主張しています。

*   ジョン：午後３時から５時まで図書館で本を読んでいた。
*   メアリー：午後４時から６時まで美容院にいた。
*   ピーター：午後３時半から４時半までジョギングをしていた。

宝石が盗まれたのは午後４時半頃です。しかし、監視カメラには、ある容疑者が午後４時20分に富豪の家から出てくる姿が映っていました。

犯人は誰でしょうか？

**第4問：不思議なケーキ**

あなたはパティシエ。お客さんから「特別なケーキ」の注文を受けました。そのケーキは、

*   必ず３つの味がすること
*   甘い味、酸っぱい味、苦い味がそれぞれ均等に感じられること
*   見た目は完全に均一で、どこを切っても同じ割合で３つの味がすること

という条件があります。

一体どんなケーキを作れば良いでしょうか？具体的な材料や作り方を述べる必要はありません。どのような構造のケーキかを答えてください。

**第5問：迷子のロボット**

あなたはプログラマー。あなたが作ったロボットが、以下の条件で動くようにプログラムされています。

*   前に３歩進む
*   右に９０度回転する
*   後ろに２歩下がる
*   左に９０度回転する

ロボットをある地点からスタートさせ、このプログラムを３回繰り返したところ、ロボットはスタート地点の真後ろにいました。

最初のスタートでロボットはどちらを向いていたでしょうか？ 東、西、南、北で答えてください。

## 解答

**第1問：魔法の鏡**

鏡がクリアになるのは「今日は晴れている」と言った時。

**解説:**

*   「私は嘘つきだ」：もしこれが真実なら、発言は嘘になるので矛盾。もしこれが嘘なら、発言は真実になるので矛盾。
*   「今日は晴れている」：今日は晴れているかどうかは、発言内容と関係なく事実なので、鏡は事実を映し出す。
*   「この謎は簡単だ」：謎が簡単かどうかは主観的な判断なので、真実を映す鏡は反応しない。

**第2問：古井戸の暗号**

数列は「HELP」を表しています。

**解説:**

*   各数字に対応するアルファベットを当てはめると、8 = H, 5 = E, 12 = L, 16 = P となります。

**第3問：盗まれた宝石**

犯人はピーター。

**解説:**

*   ピーターのアリバイは午後３時半から４時半までジョギング。
*   宝石が盗まれたのは午後４時半頃。
*   監視カメラには、ある容疑者が午後４時20分に富豪の家から出てくる姿が映っていた。
*   ジョギングから帰宅し、犯行を行う時間があるのはピーターだけ。

**第4問：不思議なケーキ**

ケーキは、３つの味が層状に重なった構造である必要があります。

**解説:**

*   ３つの層（甘い、酸っぱい、苦い）が均等な厚さで積み重なっており、どこを切っても３つの層が同じ割合で現れるようにします。
*   例えば、バウムクーヘンのように同心円状に３つの層を重ねることも可能です。

**第5問：迷子のロボット**

ロボットは最初に南を向いていた。

**解説:**
各動きを座標で考えます。
最初は北を向いているとして、座標の変化を計算します。
(0, 0)からスタート
北を向いている場合
1回目: (0, 3) -> (3, 3) -> (3, 1) -> (1, 1)
2回目: (1, 4) -> (0, 4) -> (0, 2) -> (2, 2)
3回目: (2, 5) -> (-1, 5) -> (-1, 3) -> (1, 3)

東を向いている場合
1回目: (3, 0) -> (3, -3) -> (1, -3) -> (1, -1)
2回目: (4, -1) -> (4, -2) -> (2, -2) -> (2, 0)
3回目: (5, 0) -> (5, 3) -> (3, 3) -> (3, 1)

南を向いている場合
1回目: (0, -3) -> (-3, -3) -> (-3, -1) -> (-1, -1)
2回目: (-1, -4) -> (0, -4) -> (0, -2) -> (-2, -2)
3回目: (-2, -5) -> (1, -5) -> (1, -3) -> (-1, -3)
３回繰り返すと、スタート地点(0,0)の真後ろ(0, -6)にいれば良い。

西を向いている場合
1回目: (-3, 0) -> (-3, 3) -> (-1, 3) -> (-1, 1)
2回目: (-4, 1) -> (-4, 0) -> (-2, 0) -> (-2, 2)
3回目: (-5, 2) -> (-5, -1) -> (-3, -1) -> (-3, 1)

南を向いている場合、上記を３回繰り返すと、おおよそスタート地点の後ろになる事から南向き。
(0,-3) -> (-3,-3) -> (-3,-1) -> (-1,-1)
(-1,-4) -> (0,-4) -> (0,-2) -> (-2,-2)
(-2,-5) -> (1,-5) -> (1,-3) -> (-1,-3)

```python
def solve():
    # 初期位置と向き
    x, y = 0, 0
    direction = 2  # 0: 北, 1: 東, 2: 南, 3: 西
    
    for _ in range(3):
        # 前に3歩進む
        if direction == 0:
            y += 3
        elif direction == 1:
            x += 3
        elif direction == 2:
            y -= 3
        else:
            x -= 3
        
        # 右に90度回転
        direction = (direction + 1) % 4
        
        # 後ろに2歩下がる
        if direction == 0:
            y -= 2
        elif direction == 1:
            x -= 2
        elif direction == 2:
            y += 2
        else:
            x += 2
        
        # 左に90度回転
        direction = (direction - 1 + 4) % 4
    
    print(f"最終座標: ({x}, {y})")
    
    # 最終位置がスタート地点の真後ろかどうかをチェック
    if x == 0 and y < 0:
        print("ロボットは最初に南を向いていました。")
    else:
        print("ロボットは最初に南を向いていませんでした。")
```
