## 毎日楽しめる謎解き問題：秘密のレシピ

あなたは、亡くなったおばあちゃんの家を整理しています。おばあちゃんは料理が大好きで、たくさんのレシピを残していました。その中に、ひときわ古びたレシピノートがあります。表紙には「秘密のレシピ」と書かれており、中にはいくつかの料理名と、暗号のようなものが書かれています。

**ページ1：魔法のスープ**

材料：
*   赤い野菜：X + 2
*   黄色い野菜：Y - 1
*   緑の野菜：Z / 2
*   透明な液体：1 リットル

暗号：
X = 「太陽」の画数
Y = 「月」の画数
Z = 「星」の画数

**ページ2：虹色のクッキー**

材料：
*   小麦粉：A
*   砂糖：B
*   卵：3個
*   バター：C

暗号：
A = 12 - B
B = C + 1
C = 2

**ページ3：秘密のメモ**

「真実の味は、隠された数字の中に。」

おばあちゃんの秘密のレシピを解き明かし、それぞれの料理に必要な材料の数を明らかにしてください。

## 解答

**魔法のスープ**

*   赤い野菜：X + 2 = 4 + 2 = 6個
*   黄色い野菜：Y - 1 = 4 - 1 = 3個
*   緑の野菜：Z / 2 = 9 / 2 = 4.5個 → 4個（野菜の数を考えると、切り捨てが妥当）
    *   X = 「太陽」の画数 = 4画
    *   Y = 「月」の画数 = 4画
    *   Z = 「星」の画数 = 9画

**虹色のクッキー**

*   小麦粉：A = 12 - B = 12 - 3 = 9
*   砂糖：B = C + 1 = 2 + 1 = 3
*   卵：3個（記載通り）
*   バター：C = 2

A = 9
B = 3
C = 2

**よって、虹色のクッキーの材料は**

* 小麦粉 9
* 砂糖 3
* 卵 3
* バター 2

**真実の味**

各ページで使われた数字を並べると6349332となる。
この数字は、おばあちゃんの家の電話番号の下七桁かもしれない。
