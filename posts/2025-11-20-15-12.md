## 今日の謎解きチャレンジ！

**1. 論理パズル：消えたお菓子**

4人兄弟の家に、お母さんがみんなのためにクッキーを10枚買ってきました。

*   一郎は、二郎より1枚多くクッキーを食べました。
*   二郎は、三郎より1枚多くクッキーを食べました。
*   三郎は、四郎より1枚多くクッキーを食べました。

誰もクッキーを残さず、全員が少なくとも1枚は食べたとき、四郎は何枚クッキーを食べたでしょうか？

**2. ストーリー謎解き：秘密のメッセージ**

あなたは、古い屋敷の探検家です。長い廊下の奥で、埃をかぶった古い日記を見つけました。日記には、暗号のような文章が書かれています。

"3つの鍵が眠る場所、最初の鍵は時の流れを知る者の傍ら、二つ目の鍵は静寂を愛する者の住処、最後の鍵は光を操る者の手の中。"

屋敷の中には、以下のものがあることがわかっています。

*   大きな古時計
*   図書館
*   ステンドグラスの窓

3つの鍵はそれぞれどこにあるでしょうか？

**3. 図形パズル：分割パズル**

以下の図形を、線に沿って4つの同じ形と大きさに分割してください。

```
+---+---+
|   |   |
+---+---+
|   |   |
+---+---+
```

## 解答

**1. 論理パズル：消えたお菓子**

四郎が食べたクッキーの枚数を x とすると、

*   三郎：x + 1 枚
*   二郎：x + 2 枚
*   一郎：x + 3 枚

合計で10枚なので、

x + (x + 1) + (x + 2) + (x + 3) = 10

4x + 6 = 10

4x = 4

x = 1

したがって、四郎は1枚クッキーを食べました。

**2. ストーリー謎解き：秘密のメッセージ**

*   最初の鍵は、時の流れを知る者、つまり古時計の傍らにあります。
*   二つ目の鍵は、静寂を愛する者、つまり図書館にあります。
*   最後の鍵は、光を操る者、つまりステンドグラスの窓の中に隠されています。ステンドグラスは光を屈折させ、色鮮やかな模様を作り出すからです。

**3. 図形パズル：分割パズル**

```
+---+---+
| A | B |
+---+---+
| C | D |
+---+---+
```

A, B, C, D のように分割すると、すべて同じ形と大きさになります。
