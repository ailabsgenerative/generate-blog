**第2問：消えた宝石**

あなたは名探偵。ある宝石店のオーナーから、宝石が盗まれたという依頼を受けました。現場の状況は以下の通りです。

*   宝石は店内の金庫に保管されていた。
*   金庫はダイヤル式の鍵で、オーナーしか暗証番号を知らない。
*   金庫にはこじ開けられた跡はなく、暗証番号を入力して開けられたようだ。
*   事件当日、店にはオーナー、店員のAさん、清掃員のBさんの3人がいた。
*   オーナーは事件当時、事務所で書類を作成していたと証言。
*   Aさんは接客をしていたと証言。
*   Bさんは店の奥の倉庫を掃除していたと証言。
*   防犯カメラには、Aさんがダイヤルを操作している様子が映っていた。
*   しかし、Aさんは「暗証番号は知らない」と主張している。

犯人は誰で、どのようにして金庫を開けたのでしょうか？

**第3問：迷子のロボット**

あなたはロボットプログラマー。あなたが開発したロボットが、複雑な迷路の中で迷子になってしまいました。ロボットには以下の命令を送ることができます。

*   前進（F）
*   右に90度回転（R）
*   左に90度回転（L）

ロボットは、迷路の入り口からスタートし、出口を目指します。迷路の構造は以下の通りです。

```
#########
#S      #
# ####  #
# #   # #
# # E # #
#     # #
#########
```

Sはスタート地点、Eは出口を示します。#は壁です。
最短経路でロボットを出口に導く命令の順番を答えてください。

## 解答

**第1問：数字の暗号**

法則は、前の2つの数字を足し合わせると次の数字になる、というフィボナッチ数列です。

21 + 13 = 34

よって、？に入る数字は34です。

**第2問：消えた宝石**

犯人は店員のAさんです。

Aさんは防犯カメラに映っていたにも関わらず、「暗証番号は知らない」と証言しています。つまり、Aさんは暗証番号を知らなくても金庫を開けられる方法を知っていたことになります。

ダイヤル式の金庫は、ダイヤルの回転具合を注意深く観察することで、暗証番号を推測できる場合があります。Aさんは接客中を装いながら、オーナーが金庫を開ける様子を見て暗証番号を推測し、宝石を盗んだと考えられます。

**第3問：迷子のロボット**

最短経路でロボットを出口に導く命令の順番は以下の通りです。

F, F, F, F, R, F, F, L, F, F, R, F

（解説：スタート地点から前進4回、右に回転、前進2回、左に回転、前進2回、右に回転、前進1回でゴールに辿り着きます。）
