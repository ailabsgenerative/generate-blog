## 毎日謎解き：消えた王冠と秘密のメッセージ

**物語**

あなたは腕利きの探偵。ある日、一国の王室から緊急の依頼が舞い込みました。

「王室に代々伝わる大切な王冠が盗まれたのです！犯人は大胆にも、私たちに謎を解かなければ王冠の場所は永遠に分からない、というメッセージを残していきました。」

王室が保管していたのは、犯人が残したと思われる一枚の紙。そこには奇妙な記号と数字が書かれていました。

**問題**

紙に書かれていたのは以下の通りです。

```
〇□△＝3
△△△＝9
□□〇＝8

王冠の隠し場所は？

〇□△□〇△□
```

この記号と数字から、王冠の隠し場所を解き明かしてください。

**ヒント**

*   それぞれの記号が表す数字を特定しましょう。
*   計算問題として捉えてみましょう。
*   王冠の隠し場所は、記号を数字に置き換えたものです。

## 解答

まず、記号が表す数字を特定します。

*   〇□△＝3　→　〇 + □ + △ = 3
*   △△△＝9　→　△ + △ + △ = 9

2番目の式から、△ = 3 であることが分かります。

これを1番目の式に代入すると、〇 + □ + 3 = 3 となり、〇 + □ = 0 となります。
ここで、3番目の式に注目します。

*   □□〇＝8　→　□ + □ + 〇 = 8

〇 + □ = 0 であるから、□ + □ + 〇 = □ + (□ + 〇) = □ + 0 = □ = 8となります。
〇 + □ = 0 より、〇 = -8となります。

したがって、
〇 = -8
□ = 8
△ = 3

王冠の隠し場所は、〇□△□〇△□ を数字に置き換えたものなので、

-8 8 3 8 -8 3 8となります。
したがって、王冠の隠し場所は「-8838-838」です。
