## 毎日謎解き：消えた宝石の謎

**物語**

ある日、名高い宝石商、ミスター・スミスのコレクションから、伝説の宝石「オーロラの涙」が盗まれました。警察の捜査が難航する中、ミスター・スミスは、腕利きの探偵であるあなたに助けを求めました。

現場に残されていたのは、奇妙な暗号が書かれた1枚のカードだけでした。

**暗号**

`3, 1, 2, 4`

`A, B, C, D`

`C, A, D, B`

`B, D, A, C`

`D, C, B, A`

**手がかり**

*   ミスター・スミスの証言：「オーロラの涙は、私のコレクションの中で最も価値のある宝石でした。犯人は、宝石の価値を知っている者でしょう。」
*   ミスター・スミスのコレクションは、宝石の種類ごとに整理されており、それぞれアルファベットでラベル付けされています。
    *   A: ダイヤモンド
    *   B: ルビー
    *   C: サファイア
    *   D: エメラルド
*   ミスター・スミスのコレクションケースは、宝石の種類ごとに４つの引き出しに整理されており、それぞれの引き出しには１つづつ宝石が入っている。

**問題**

暗号を解読し、オーロラの涙が隠されている場所を特定してください。

## 解答

暗号は、各アルファベット（A, B, C, D）に対応する宝石の並び順を表しています。最初の数字の羅列 (3, 1, 2, 4) は、各行の文字を並び替える順番を示しています。

つまり、

1行目: `C, A, D, B` → `A, B, C, D` (1, 2, 3, 4番目の文字を、3, 1, 2, 4番目に並び替える)
2行目: `B, D, A, C` → `D, B, A, C` (1, 2, 3, 4番目の文字を、3, 1, 2, 4番目に並び替える)
3行目: `D, C, B, A` → `C, D, B, A` (1, 2, 3, 4番目の文字を、3, 1, 2, 4番目に並び替える)

となります。

したがって、暗号を解読すると、

*   1行目: A (ダイヤモンド)
*   2行目: D (エメラルド)
*   3行目: C (サファイア)

となり、犯人はダイヤモンド、エメラルド、サファイアのコレクションケースの中にオーロラの涙を隠したことになります。

ミスター・スミスの証言「オーロラの涙は、私のコレクションの中で最も価値のある宝石でした。犯人は、宝石の価値を知っている者でしょう。」より、宝石の価値を知っている内部の犯行である可能性が高いと考えられます。
